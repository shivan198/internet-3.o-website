Web 3.0

Web 3.0 has no particular definition but it is rather referred to as the new evolution of the web, its third generation, and includes specific innovations and practices. Or web 3.0 is where computers generate more information than humans.

Web 3.0 can rather be explained on its 5 main features:
  1. Semantic Web
  2. Artificial Intelligence
  3. Connectivity
  4. 3D Graphics
  5. Ubiquity
Semantic Web

The main functionality of the semantic web is understanding through words, not by keywords or numbers. semantic can explain the meaning and emotion of the words and artificial intelligence can utilize the data to retrieve the information. The main idea is to create a spider web of knowledge across the internet which will help it to understand the meaning of words to generate, share, and connect content through search and analysis. Tim Berners Lee also stated that “Web of data that can be processed by words”.

Artificial Intelligence

By combining Artificial Intelligence with natural language processing, computers can understand information like humans and provide faster and more relevant results to the users. They become more intelligent to satisfy the needs of users.

Connectivity

With the help of semantic metadata information is more connected as a result users can experience another level of connectivity that leverages all the available information. As per now, the data is controlled by the application but by the connectivity feature, we can control the data from the whole web. Example - controlling google docs data from google calendar.

3D Graphics

The use of three dimensional images increases widely in web 3.0, the services like computer games, eCommerce, images in websites will be in 3D graphics. Most of the researchers also say that every person has a 3d virtual image which will be as popular as email address and phone number. Example - 3d virtual image in games.

Ubiquity

ubiquity means the presence of everywhere and anywhere at the same time. By using ubiquity content can be accessible by multiple applications, every device is connected to the web, the services can be used everywhere. Example - posts on social media.

Decentralization

Web 3.0 assures with decentralized data, where data is stored as peer to peer connectivity, not as centralized data

Centralized data is where the data is stored at one place, we can get the data from one place.

Example - AWS, Google cloud, etc.

PROS CONS
By the use of the semantic web the data is more connected and linked. Devices which are less advanced will not be able to handle Web 3.0.
Most efficient searching Web 1.0 websites will be out of date.
Marketing will be done in a better way. It can be very complicated for newcomers to understand.
More efficient web browsing.
Effective communication.
Examples

If I type as “I want to watch a horror movie and then go for a Chinese restaurant” then web 3.0 will understand your emotions and search relevant data and will some you the nearby horror movie theaters and also the Chinese restaurants near you.

When you and your friend search for cricket, web 3.0 will show different results for two people based on parameters like favorite cricketer, shots, stadium, and series by these people sitting at the same location will also get a different location.
